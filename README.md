This is a Pokedex, based on the original Pokedex. It can search a pokemon based on the id or name provided, and will get information on them. It grabs the information from the PokeAPI, and the images from a related repository. You install the APK found in the bin/release/ folder to run it for Android, and for windows you enter the windows, your version of windows, AppPackages, the desired version, and click on the msix file, and you will use that to open it later too.
